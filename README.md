# Анализ покупательского поведения

## Описание проекта
Проект содержит анализ данных о покупках 1375 клиентов. Исследуются различные аспекты покупательского поведения, включая демографические характеристики, предпочтения в покупках и закономерности в совершении покупок.

## Структура данных
Датасет содержит следующие колонки:
- **Customer ID** - уникальный идентификатор клиента
- **Age** - возраст клиента
- **Gender** - пол клиента
- **Item Purchased** - купленный товар
- **Category** - категория товара
- **Purchase Amount (USD)** - сумма покупки в долларах
- **Location** - местоположение клиента
- **Size** - размер товара
- **Color** - цвет товара
- **Season** - сезон покупки
- **Review Rating** - рейтинг отзыва
- **Subscription Status** - статус подписки
- **Shipping Type** - тип доставки
- **Discount Applied** - применена ли скидка
- **Promo Code Used** - использован ли промокод
- **Previous Purchases** - количество предыдущих покупок
- **Payment Method** - метод оплаты
- **Frequency of Purchases** - частота покупок

## Методы анализа
1. **Предобработка данных**:
   - Проверка пропущенных значений
   - Поиск и удаление дубликатов
   - Анализ выбросов

2. **Исследовательский анализ (EDA)**:
   - Статистическое описание данных
   - Визуализация распределений
   - Анализ корреляций

3. **Группировка и агрегация**:
   - Анализ по категориям товаров
   - Исследование сезонных закономерностей
   - Анализ по демографическим признакам

## Результаты
Основные выводы из анализа:
1. Средний возраст покупателей - 45 лет
2. Наибольшее количество покупок приходится на категорию "Clothing"
3. Наиболее популярный сезон для покупок - Winter
4. Средняя сумма покупки составляет $59.7
5. Наблюдается слабая корреляция между возрастом и суммой покупки
